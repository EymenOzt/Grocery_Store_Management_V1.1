# Security Policy

## Supported Versions

The following table shows which versions of this project are currently supported with security updates.

| Version        | Security Support |
|---------------|------------------|
| Latest        | ✅ Supported     |
| Older versions| ❌ Not supported |

Only the latest version receives security fixes and updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately using one of the following methods:
- GitHub Security Advisories (preferred)
- Email (if provided in the repository)

Please include:
- A clear description of the vulnerability
- Steps to reproduce (if available)
- Possible impact

I will review the report and respond within a reasonable timeframe.

## Support & Maintenance

This project is maintained by a **single developer**.

| Support Type        | Status |
|--------------------|--------|
| Security fixes     | ✅ Yes |
| Bug fixes          | ⚠️ Best effort |
| Feature requests   | ⚠️ Limited |
| Long-term support  | ❌ No |

Support is provided on a best-effort basis and depends on availability.

## Responsible Disclosure

Please allow time for the issue to be investigated and resolved before any public disclosure.
