# Contributing

Thank you for your interest in contributing to this project!

This project is maintained by a single developer, and contributions are welcome
on a best-effort basis.

## How to Contribute

You can contribute in the following ways:
- Reporting bugs
- Suggesting improvements
- Fixing issues
- Submitting pull requests

## Guidelines

- Be respectful and follow the Code of Conduct
- Keep code clean and readable
- Test your changes before submitting
- Write clear and descriptive commit messages

## Pull Requests

- Make sure your changes are relevant to the project
- Avoid unrelated or large changes in a single pull request
- Describe clearly what your pull request does

## Issues

Please use GitHub Issues to report bugs or request features.

For security-related issues, please follow the instructions in `SECURITY.md`.
